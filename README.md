# Test Projects

Repository for Test Projects

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy the project on a live system.

### Prerequisites

SpecFlow Visual Studio Extension

### Installing

Install Visual Studio's SpecFlow extension to have the test steps display correctly.
Build the solution to map the steps in the feature files to the Steps files

Change the Environment in Configuration\Configuration.json to the required server name
